Formatting Help

On this page you can find a guide on how to use our online "IDE".
You can use it to write your own code and see the results.

You can also use the IDE to embed code on your personal website, it is licensed under the MIT License!

In order to use the embedding feature you should know about the parameters:

  • embed - can either be 1 or 0, used to hide file options (should be 1 when embedding in tutorials)
  • file - the file to embed (stored in /examples/FILENAME.js)
  • outlines - how many lines the output should show.
  • disabled - when 1 or true, the textarea will not allow input
