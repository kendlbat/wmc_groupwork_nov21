Einführung in JavaScript

JavaScript ist, wie der Name schon vermuten lässt, eine Skriptsprache. Das heißt, dass die Codezeilen nacheinander von einem Interpreten ausgeführt werden. Wenn im Code ein Fehler sein sollte, dann sieht man diesen erst, nachdem die vorherigen Zeilen ausgeführt wurden.

JavaScript ist die fünftbeliebteste Programmiersprache der Welt, welche zumeist für die Funktionalität als auch für das dynamische Design einer Webseite verwendet wird. Ein weiterer Grund für die Beliebtheit der Sprache, ist, dass sie für jedes Gerät und für jeden Browser universal einsetzbar ist. Man muss nicht für andere Geräte andere Programmiersprachen lernen, sondern alle Webseiten können mit JavaScript geschrieben werden und die Funktionen sind auf allen Geräten gleich.
