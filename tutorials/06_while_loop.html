While-Loop

Die While-Schleife ist eine Schleife, welche eine unbestimmte Anzahl an Anläufen durchgeht. Ein Beispiel für eine While-Schleife sieht so aus.

Beispielcode 1:

Output: Alle Zahlen von 0 bis 9

Dabei ist der Code innerhalb der runden Klammer (//condition) die Bedingung, die am Anfang oder nach einem Ablauf abgerufen wird, um zu bestimmen, ob noch ein Ablauf durchgegangen werden soll. (Der Code innerhalb der geschwungenen Klammern)

Do-While-Loop

Die do-while-Schleife ist vom Aufbau und von der Funktionalität sehr ähnlich zur While-Schleife. Diese sieht zum Beispiel so aus:

Beispielcode 2:

Output: Alle Zahlen von 0 bis 9

Der einzige Unterschied besteht darin, dass die Do-While-Schleife mindestens einen Ablauf hat, während die While-Schleife, wenn die Bedingung schon am Anfang false ausgibt, einfach übersprungen wird.
