Funktionen

Eine Funktion ist ein Codeblock, welcher bestimmte Aufgaben erfüllt.

Beispielcode 1:

Output: 5

Syntax einer Funktion

Eine Funktion wird mit dem function Schlüsselwort definiert, dabei werden innerhalb der runden Klammern die Parameter übergeben, welche dann in der Funktion verwendet werden. die Funktion selber wird innerhalb der geschwungenen Klammern geschrieben

function nameDerFunktion (parameter1, parameter2, ...) {

//code

}

Wenn man eine Funktion aufrufen möchte, muss man die vorgegebenen Parameter mitgeben. Danach wird der in der Funktion enthaltene Code aufgerufen. Falls die Funktion mit return endet, dann wird der danach stehende Wert übergeben, entweder an eine Variable, oder an eine andere Funktion.

Warum sollte man Funktionen verwenden?

Funktionen können wiederverwendet werden, das heißt, dass man, wenn sich ein Stück Code wiederholt, wird man diesen nicht etliche Male schreiben, sondern in einer Funktion auslagern, und diese Funktion dann ausrufen.
